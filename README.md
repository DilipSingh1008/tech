# Tech Admin Panel

Tech Admin Panel is a scalable and reusable admin dashboard built using the MERN stack.
It is designed to manage multiple modules of a company website such as users, products, services, blogs, media, and more.

The system is divided into three main parts:

1. Admin Dashboard
2. Backend API
3. Client Website

The admin panel is built in a generic way so it can be reused across different projects.

## Features

* Authentication (Login / Register / Forgot Password)
* Dashboard Analytics
* Location Management (State & City)
* Category & Subcategory Management
* Product Management
* Service Management
* Blog Management
* News Management
* CMS Page Management
* FAQ Management
* Banner Management
* Client Management
* Vendor Management
* Career / Job Management
* Media & Media Post Management
* Role & Permission Management
* User Management
* Enquiry Management
* Site Settings

## Tech Stack

Frontend:

* React.js
* React Router
* Context API

Backend:

* Node.js
* Express.js
* REST API

Database:

* MongoDB

## Architecture

The project follows a modular architecture where the admin panel interacts with the backend API to manage dynamic content used by the client website.

## Purpose

The goal of this project is to create a reusable admin dashboard that can be integrated with multiple company websites.
